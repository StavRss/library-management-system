# Library Management System (C++)

## Περιγραφή
Πανεπιστημιακή εργασία 1ου εξαμηνου σε C++ για διαχείριση βιβλίων και αντιτύπων.

## Λειτουργίες
- Προσθήκη βιβλίων
- Αναζήτηση βιβλίων
- Διαχείριση αντιτύπων

## Τεχνολογίες
- C++
- Object-Oriented Programming
- Linked Lists
